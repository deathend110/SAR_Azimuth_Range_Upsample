# 阶段0：BRAU补全实验设计

## 0. 目标说明

本文阶段0实验的目标，是围绕以下三个核心创新点补齐证据链，并形成可直接进入会议论文实验部分的结果材料。

### 核心创新点

1. 提出一种面向 1-bit SAR 成像的距离向-方位向双向 `RxAx` 上采样框架。
2. 揭示双向拆分整体优于单向上采样，且严格平方根均衡并非必要条件。
3. 提出并验证一种具有鲁棒性的双向拆分设计准则，即在不同 `RT` 阈值场构造方式、不同 `As` 参数以及不同 `Q` 值下，“双向拆分优先”的规律仍然稳定成立。

### 本文档使用方式

- 本文档按“先统一设置，再逐组实验，再统一整理结果”的顺序编排。
- 每一项任务均带有勾选框，完成后可直接勾选。
- 每组实验都写明了：
  - 实验目的
  - 输入设置
  - 操作步骤
  - 必须产出
  - 对应支撑的创新点

## 3. 实验一：主结果实验

### 3.1 实验目的

验证以下命题：

- 双向 `RxAx` 上采样整体优于单向上采样。
- 在固定总上采样预算 `Q` 下，是否严格接近平方根均衡不是必要条件。

### 3.2 基线设置

对每个 `Q`，至少包含以下四类统一对照：

- [ ] `NoUpsample`：无上采样基线
- [ ] `RangeOnly`：纯距离向上采样，即 `R=Q, A=1`
- [ ] `AzimuthOnly`：纯方位向上采样，即 `R=1, A=Q`
- [ ] `Bidirectional`：双向拆分上采样，即 `R>1, A>1`

### 3.3 具体执行步骤

1. [ ] 对每个 `Q` 自动枚举所有满足 `R * A = Q` 的整数因子组合。
2. [ ] 固定使用 `SplitRT + As=0.6` 跑第一版主实验。
3. [ ] 在每个数据集、每个随机种子下，对所有组合计算 `PSNR` 和 `SSIM`。
4. [ ] 对每个 `Q` 统计每个 `R×A` 组合的均值和标准差。
5. [ ] 标记每个 `Q` 下的最佳单向组和最佳双向组。

### 3.4 必须产出

- [ ] 每个 `Q` 一张 `PSNR` 图
- [ ] 每个 `Q` 一张 `SSIM` 图
- [ ] 一份汇总表：每个 `Q` 下所有 `R×A` 组合的均值、标准差
- [ ] 一份结论表：每个 `Q` 下最佳单向组、最佳双向组及其差值

### 3.5 需要重点观察的问题

- [ ] 小 `Q` 时是否“越接近平衡越好”

## 4. 实验二：RT 阈值消融实验

### 4.1 实验目的

本实验不再局限于 `RT` 家族内部对比，而是扩展为“阈值量化策略泛化实验 + RT 家族内部消融实验”两层结构，目标是回答以下三个问题：

- 双向 `RxAx` 上采样的优势，是否在不同 1-bit 阈值量化策略下都成立。
- 相比 `ZT / NCT / GT`，结构化 `RT` 是否能进一步稳定放大双向拆分的收益。
- 在 `RT` 家族内部，`FullRT` 与 `SplitRT` 是否只影响细节，而不改变“双向拆分优先”的主趋势。

### 4.2 实验设置

建议选取代表性 `Q`：

- [ ] `Q=4`
- [ ] `Q=12`
- [ ] `Q=24`
- [ ] `Q=36`

### 4.3 阈值策略定义

本实验统一比较以下四类阈值量化策略：

- [ ] `ZT`：零阈值 1-bit 量化，对应 `NoRT, As=0`
- [ ] `NCT`：全局固定阈值 1-bit 量化
- [ ] `GT`：全局零均值高斯随机阈值场
- [ ] `RT`：本文方法族，包括 `FullRT` 与 `SplitRT`

### 4.4 各阈值策略的实验角色

- [ ] `ZT` 作为最基础的 1-bit 量化基线
- [ ] `NCT` 作为固定阈值基线，用于排除“任意非零阈值都有效”的解释
- [ ] `GT` 作为无结构随机阈值基线，用于排除“只要引入随机噪声阈值就足够”的解释
- [ ] `RT` 作为结构化随机阈值方法，用于体现本文方法的优势

### 4.5 参数匹配原则

为保证比较公平，不同阈值策略必须统一强度参考尺度。

- [ ] 所有阈值强度统一参考同一个全局尺度 `sigma_ref`
- [ ] `NCT` 设为 `T = c * sigma_ref`
- [ ] `GT` 设为 `T(i,j) ~ N(0, (c * sigma_ref)^2)`
- [ ] `RT` 设为 `A_rt = As * sigma_ref`
- [ ] `ZT` 视为 `c=0` 或 `As=0` 的零阈值特例，但在实验中单独列为一类基线

说明：

- `sigma_ref` 必须来自同一套统一定义，不能每类阈值各用一套尺度。
- `GT` 与 `RT` 的比较重点应建立在“强度匹配”而不是“绝对数值大小不同”上。

### 4.6 每个 `Q` 下建议保留的 `RxAx` 组

为防止图过乱，每个 `Q` 下优先保留以下四类：

- [ ] 最佳纯距离向组
- [ ] 最佳纯方位向组
- [ ] 近均衡双向组
- [ ] 非均衡双向组

### 4.7 子实验 2A：阈值量化策略横向对比实验

#### 4.7.1 子实验目的

验证“双向拆分优先”的结论是否跨阈值量化策略成立，并突出结构化 `RT` 相对于 `ZT / NCT / GT` 的优势。

#### 4.7.2 第一轮最小对比设置

先使用固定代表参数做第一轮轻量实验，避免实验量过大。

- [ ] `ZT`
- [ ] `NCT(c=0.6)`
- [ ] `GT(c=0.6)`
- [ ] `SplitRT(As=0.6)`

如果第一轮结果已经清楚显示主趋势成立，再进入第二轮细化扫描。

#### 4.7.3 具体执行步骤

1. [ ] 对每个代表性 `Q`，固定一组比较用的 `R×A` 组合。
2. [ ] 在每个 `Q` 下，对上述四类阈值策略分别运行纯距离向组、纯方位向组、近均衡双向组和非均衡双向组。
3. [ ] 统计各组的 `PSNR` 和 `SSIM` 均值、标准差。
4. [ ] 计算“最佳双向组 - 最佳单向组”的增益量，作为双向收益增量。
5. [ ] 比较不同阈值策略下，双向收益增量是否稳定为正。

#### 4.7.4 第二轮细化扫描设置

当第一轮结果明确后，再做参数扫描：

- [ ] `NCT`：扫描 `c = 0:0.1:1`
- [ ] `GT`：扫描 `c = 0:0.1:1`
- [ ] `RT`：扫描 `As = 0:0.1:1`

说明：

- `ZT` 对应于 `c=0`，可在图例中保留单独标识。
- 若实验量过大，可先只扫描 `GT` 与 `RT`，把 `NCT` 保留为代表参数版本。

#### 4.7.5 必须产出

- [ ] 每个 `Q` 一张 `PSNR` 柱状图，横轴为 `RxAx` 组，组内柱子为不同阈值策略
- [ ] 每个 `Q` 一张 `SSIM` 柱状图
- [ ] 一份双向收益增量表：记录不同阈值策略下“最佳双向组 - 最佳单向组”的差值
- [ ] 一份阈值策略汇总表：比较 `ZT / NCT / GT / RT` 的整体表现

#### 4.7.6 需要重点观察的问题

- [ ] 双向组是否在 `ZT / NCT / GT / RT` 下都普遍优于单向组
- [ ] `NCT` 是否只能提供有限改善，说明“任意非零阈值”并不足够
- [ ] `GT` 是否虽然也有效，但整体不如 `RT` 稳定
- [ ] `RT` 是否在不同 `Q` 下更稳定地保持双向优势
