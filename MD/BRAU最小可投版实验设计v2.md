# BRAU 最小可投版实验设计 v2

> 基于 Socratic Layer 1 讨论结论重新设计，聚焦单一核心claim。

## 核心Claim

**在1-bit SAR成像中，将固定上采样预算Q拆分到距离向和方位向两个维度（R×A = Q），比集中到单一维度的上采样更有效。**

佐证层（不独立成贡献）：√平衡不必要 | 非整数有效 | SplitRT≈FullRT鲁棒

## 实验一：主结果实验

### 目的

用统一的四类基线对照，系统性验证核心claim：双向拆分稳定优于单向。

### 四类统一对照

对每个Q，必须包含：

| 类别 | 定义 | 示例(Q=4) |
|------|------|-----------|
| NoUpsample | R=1, A=1, 直接1-bit量化无上采样 | R1A1 |
| RangeOnly | R=Q, A=1, 纯距离向上采样 | R4A1 |
| AzimuthOnly | R=1, A=Q, 纯方位向上采样 | R1A4 |
| Bidirectional | R>1 且 A>1, 双向拆分 | R2A2 |

### 实验条件

**整数Q组**（验证基本规律）：

- Q_list = [4, 6, 8, 9, 10]
- 对每个Q枚举全部整数因子对 R×A = Q
- 固定使用 SplitRT, As = 0.6
- 注：1-bit SAR场景下超高上采样率不现实，Q上限设为10

**非整数Q组**（佐证：优势不是整数分配的巧合）：

- 选取3-4个非整数因子组合，例如：
  - Q=3: R1.5 × A2
  - Q=4.5: R1.5 × A3
  - Q=6: R2 × A3（整数，但非均衡）
  - Q=5: R2 × A2.5 或 R2.5 × A2
- 与同等总Q的纯方位向和纯距离向对比

### 数据与统计设置

- 数据集：7个SAR数据集全用（Bangkok, city1, city2, SAR_figure, filed, port, suburb）
- 每个数据集分层抽样 10 帧，共 70 个测试样本
- 随机种子：5个（seed = 2026, 2027, 2028, 2029, 2030）
- 每个种子独立跑一遍，最终取 5-seed 平均
- 统计量：PSNR/SSIM 的 mean ± std（跨 70×5 = 350 个观测）
- **配对 Wilcoxon signed-rank test**：对每个Q下 "最佳双向组 vs 最佳单向组" 做显著性检验，报告 p-value

### 必须产出

1. **主结果表**：每个Q下四类对照的 PSNR_Mean ± Std, SSIM_Mean ± Std
2. **增益表**：每个Q下 "最佳双向 - 最佳单向" 的 ΔPSNR, ΔSSIM, p-value
3. **主结果图**：横轴 Q，纵轴 PSNR/SSIM，四类对照各一条曲线，双向区域用阴影标注
4. **非整数佐证图**：3-4个非整数组合 vs 等Q单向对照的柱状图（单独一张小图，或嵌入主图）

### 对应支撑

- 核心Claim（双向 > 单向）
- 佐证：√平衡不必要（多个双向组进入plateau）
- 佐证：非整数有效（非整数组合也优于单向）

## 实验二：机制验证实验

### 目的

回答"为什么双向更有效"。本实验不把机制验证理解为“额外画几张图”，而把它定义为一次 **沿整个 SAR 成像 pipeline 的取证分析（pipeline forensics）**：定位双向上采样的优势到底是在什么节点被引入、在什么节点被保留，并最终如何体现到成像结果上。

核心待验证假设：

> SAR信号具有2D频域支撑结构。只在1D推开量化噪声，另一个维度的噪声仍压在有效带宽内。双向上采样同时在两个维度推开噪声，更好地匹配SAR的2D信号结构。

更具体地说：

- **机制主张A（来源）**：双向优势的主要来源发生在 **上采样 + 1-bit量化** 这一步，而不是后续RC/RCMC/方位聚焦“凭空创造”出来的。
- **机制主张B（形式）**：双向上采样的收益表现为 **2D支撑外量化残差更少**，即噪声在距离向和方位向两个方向都被同时推离有效带宽。
- **机制主张C（传递）**：这种更干净的2D谱结构，会在后续RC、RCMC和最终成像中被保留，从而形成更好的最终图像质量。

### 代表性案例

选取 Q=4（小Q，双向优势最明显）做深度机制分析：

- 方案1: R1A4（纯方位向）
- 方案2: R4A1（纯距离向）
- 方案3: R2A2（双向均衡）
- 对照: R1A1（无上采样）

固定1个数据集、1帧图像、seed=42。

建议优先选用：

- 一个实验一中双向增益明显、且视觉结构较丰富的样本
- 同时避免选择过于平坦或散射点极少的样本，否则频谱差异可能不稳定

### Pipeline 分解思路

结合当前代码实现，实验二应把现有链路拆成以下5个取证节点：

1. **Node-0: Upsampled Signal**  
   `signal_up = two_dim_upsample_fft(...)`  
   观察上采样本身如何改变二维频域支撑的采样密度。
2. **Node-1: Quantized Signal**  
   `channel_1bit = quantize_1bit_with_U(signal_up, U)`  
   这是最关键节点。比较不同方案下 1-bit 量化残差是如何分布到二维频域中的。
3. **Node-2: After RC**  
   `RC = Range_Compress(...)`  
   观察距离压缩后，前一阶段形成的二维噪声结构是否仍然可见。
4. **Node-3: After RCMC**  
   `RCMC_out = RCMC(...)`  
   观察距离徙动校正后，双向方案是否仍保留更低的支撑外谱泄漏。
5. **Node-4: Final Image**  
   `IMG = SAR_Imaging(...)`  
   最终验证这些中间差异是否传递为更好的视觉质量和ROI结构恢复。

其中：

- **Node-1 是机制来源判定点**
- **Node-2 / Node-3 是机制传递判定点**
- **Node-4 是机制结果判定点**

### 需保存的中间结果

对每个方案，不应只保存“图像”，而应同时保存 **复数中间结果 + 可视化所需派生量**。建议至少保存以下内容：

| 节点 | 保存内容 | 观察目的 |
|------|---------|---------|
| Node-0: Upsampled Signal | `signal_up` 本身；其2D FFT幅度谱 | 建立各方案上采样后二维支撑结构的基线 |
| Node-1: Quantized Signal | `channel_1bit`；以及 `channel_1bit - signal_up` 或其等价残差表征；2D FFT幅度谱 | 判断量化误差最初是如何被推向不同方向的 |
| Node-2: After RC | `RC`；其2D FFT幅度谱 | 观察RC后噪声结构是否仍保留 |
| Node-3: After RCMC | `RCMC`；其2D FFT幅度谱 | 观察RCMC后支撑外噪声是否仍有差异 |
| Node-4: Final Image | `IMG`、ROI、局部放大图 | 验证最终图像质量差异 |

### 建议的定量指标

实验二必须避免“只看图说话”。除2D谱图外，建议增加以下三个定量指标：

1. **1D中心剖面**
   - 从2D谱图中提取距离向中心行
   - 从2D谱图中提取方位向中心列
   - 用于直观看“哪个方向的噪声被推开了”

2. **支撑外能量占比（off-support energy ratio）**
   - 以GT链路或NoUpsample链路估计有效二维支撑区域
   - 计算 `支撑外能量 / 总能量`
   - 核心是比较 R1A4、R4A1、R2A2 在 Node-1 / Node-2 / Node-3 的该指标

3. **方向性谱泄漏指标**
   - 距离向泄漏比：比较距离方向零填充区域的平均能量
   - 方位向泄漏比：比较方位方向零填充区域的平均能量
   - 该指标用于支撑“单向只清理一个方向，双向同时清理两个方向”

### 机制图设计

**核心图：Node-1 与 Node-2 的 2D频谱对比图**

不建议只画一个节点。建议至少画两层：

- 第一层：**Node-1（1-bit后）** 的2D FFT幅度谱
- 第二层：**Node-2（RC后）** 的2D FFT幅度谱

每层都对四个方案并排展示：

```
┌─────────┬─────────┬─────────┬─────────┐
│ NoUp    │ Range   │ Azimuth │ R2A2    │
│ (R1A1)  │ (R4A1)  │ (R1A4)  │ (BiDir) │
│         │         │         │         │
│ 频谱:   │ 频谱:   │ 频谱:   │ 频谱:   │
│ 全带宽  │ 距离向  │ 方位向  │ 两个    │
│ 噪声满  │ 噪声推  │ 噪声推  │ 方向都  │
│ 压      │ 开，但  │ 开，但  │ 推开   │
│         │ 方位仍  │ 距离仍  │         │
│         │ 噪声压  │ 噪声压  │         │
└─────────┴─────────┴─────────┴─────────┘
```

如果假设正确，应看到：
- `R4A1`：距离向支撑外能量下降更明显，但方位向仍有较重泄漏
- `R1A4`：方位向支撑外能量下降更明显，但距离向仍有较重泄漏
- `R2A2`：两个方向都出现更低的支撑外能量，呈现更均衡的2D噪声推开效果
- 这种差异在Node-1最明显，在Node-2/Node-3仍然可以观察到，但可能逐步减弱

**补充图1：距离向/方位向1D剖面图**

从Node-1或Node-2的2D频谱中分别提取：

- 距离向中心行
- 方位向中心列

用于展示：

- `R4A1` 在距离向更干净
- `R1A4` 在方位向更干净
- `R2A2` 在两个方向都较干净

**补充图2：三阶段中间图总览**

这样可以直观看到：差异不是在最终图像中“突然出现”的，而是在前面节点已经埋下，并逐步传递到最终结果。

**补充表：机制指标表**

- off-support energy ratio

### 必须产出

1. Node-1 / Node-2 的2D频谱对比图（核心机制图）
2. 1D频谱剖面图（距离向 + 方位向各一张）
3. Node-2 / Node-3 / Node-4 的中间结果总览图
4. 一张机制指标小表（off-support / directional leakage）
5. 一段机制解释文字草稿（200-300字）

### 验证判据

- [ ] Node-1上，`R4A1` 的距离向泄漏低于 `R1A4`
- [ ] Node-1上，`R1A4` 的方位向泄漏低于 `R4A1`
- [ ] Node-1上，`R2A2` 在两个方向上的泄漏都低于对应单向方案
- [ ] Node-2 / Node-3 上述趋势仍然存在，哪怕幅度变弱
- [ ] Node-4 最终ROI质量排序与前述中间证据一致
- [ ] 1D剖面图与机制指标表能定量支撑上述结论

### 结果解释规则

为避免过度解释，实验二的结论应按以下规则书写：

- **若 Node-1 就能看出明显双向优势，且 Node-2 / Node-3 / Node-4 一路延续**
  则可以较强地声称：双向优势主要来源于二维量化噪声塑形。
- **若 Node-1 差异弱，但 Node-2 / Node-3 / Node-4 差异明显**
  则应谨慎表述为：双向上采样与后续SAR成像链路存在协同作用，而非单纯的前端量化塑形。
- **若只有 Node-4 有差异，而中间节点差异很弱**
  则不能强讲“2D噪声塑形机制已验证”，只能说该现象可能与后续聚焦/校正过程的非均匀放大有关。

如果验证判据不满足——即中间节点证据不足、只能在最终图像上看到差异——则需要在论文中诚实写：

> 本文观察到稳定的双向优势，但其完整物理机制尚未被充分刻画；二维量化噪声塑形是一个合理解释，但更深入的频域建模与理论分析留待未来工作。

此时论文叙事应从“强机制证明”降级为“机制线索 + 主结果统计证据”。

## 实验三：阈值鲁棒性实验

### 目的

回答：核心claim是否依赖于特定的RT阈值构造方式？

### 实验结构（精简版）

不像旧设计那样做4类阈值(ZT/NCT/GT/RT)的全消融——4页论文放不下。

**精简为两个子实验：**

**子实验3A：SplitRT vs FullRT（已部分完成）**

- 已有 Compare_R2A2_FullRT_vs_SplitRT.m 的结果
- 需补充：扩展到更多Q值（不只是R2A2），选取 Q=4, 6, 9 三个代表Q
- As扫参 0:0.1:1，每个Q下画 PSNR-As 和 SSIM-As 曲线
- 结论：SplitRT ≈ FullRT → claim对RT内部构造鲁棒

**子实验3B：ZT vs RT（最关键的外部鲁棒性检验）**

- 选取 Q=4, 6, 9 三个代表Q
- 对每个Q比较：ZT(As=0) vs SplitRT(As=0.6) 下的双向vs单向
- 只需证明：即使在ZT下，双向仍然优于单向（只是RT放大了优势）
- 不需要做NCT和GT——4页论文放不下，且ZT vs RT已经回答了核心鲁棒性问题

### 必须产出

1. FullRT vs SplitRT 的 As曲线图（3个Q各一张，或合并）
2. ZT vs RT 下双向增益对比图（1张柱状图或1张增益差值图）
3. 一段结论文字：双向优势对阈值构造方式具有鲁棒性

### 对应支撑

- 佐证：SplitRT ≈ FullRT（RT内部鲁棒）
- 佐证：ZT下双向也优于单向（RT只是放大优势，不是创造优势）
