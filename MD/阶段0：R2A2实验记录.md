# 1目前实验的问题R2A2的效果显著差于单独做上采样的，其原因排查有下列：
1. R2A2下，整体是4倍数上采样。但是方位向和距离向的4倍上采样的MIN，MAX值不同。距离上采样能赋予SAR图片更多的能量。我们需要单独计算其归一化系数或者干脆用/max归一化
2. RT阈值的问题。以方位向上采样举例。在方位向上做上采样，那么对应的RT阈值是行向量，也就是说每一列的阈值一致。距离向上采样对应的，RT阈值是列向量，每一行的阈值一致。我现在直接基于2D上采样后的信号，生成一个全局的RT阈值矩阵，其本质就类似一个高斯噪声矩阵，失去了这个随机相位阈值的特殊性

# 3 验证二维RT阈值的有效性
1. 通过设As=0和As=0.6（理论最优）有个有意思的结果是As=0下的结果反而不如As=0.6.这说明至少目前的2维随机RT是有效的。随着As的提高，R2A2的指标表现差值更多。
2. 我们做了个新的对比：针对2D的全随机RT阈值，我们现在是先构造距离向相位列向量和方位向相位行向量，再做扩展相加，R2A2依旧保留了As=0和As=0.6下全最优的特性。
3. 为了做更全面的验证，下一步是对不同底图的SAR序列做分层抽样，每个底图中抽取20帧，计算两种RT阈值生成方式在As从0-1的过程中，那种方式在R2A2下会更好

![alt text](../RT_Compare_R2A2_Output/R2A2_FullRT_vs_SplitRT_SSIM.png)

看这两张图，我们的结论是：
在固定 R2A2 上采样配置下，对比逐点独立随机的 Full 2D RT 与可分离的 Split 2D RT，对比实验表明，FullRT 与 SplitRT 的 PSNR/SSIM 曲线基本重合，SplitRT 在多数 As 取值下略高，但提升幅度很小。说明在 R2A2 配置下，二维 RT 阈值场采用 fully random 形式还是可分离形式，并不会显著改变整体成像性能；R2A2 的性能趋势对 RT 生成方式具有较好的鲁棒性。

在固定总上采样倍率 Q 的条件下，将上采样同时分配到距离向与方位向两个维度，能够稳定优于纯单向上采样。对于双向分配方案，接近平衡的 R×A 组合在小到中等 Q 下通常表现更好；但当 Q 增大后，多种双向组合的性能会收敛到相近水平。因此，对当前 1-bit SAR 成像链路而言，“二维拆分本身”比“是否严格接近平方根分配”更关键。
