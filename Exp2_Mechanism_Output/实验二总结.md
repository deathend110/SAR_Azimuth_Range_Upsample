# 实验二总结

## 1. 实验目的

实验二用于从 SAR 成像 pipeline 的中间节点出发，分析双向上采样为何有效。核心目标不是只看最终成像指标，而是观察：

1. 1-bit 量化残差在频谱中的分布方式；
2. 这种分布方式经过距离压缩（RC）、距离徙动校正（RCMC）后是否仍然保留差异；
3. 双向上采样 `R2A2` 是否在二维频谱支撑外泄漏方面更优。

本次实验固定参数如下：

- `seed = 42`
- `As = 0.600`
- `dataset = SAR_Dataset_city2_histeq`
- `file = rstart 301.mat`
- `c_start = 6500`

比较的四组方法为：

- `R1A1_NoUp`：不做上采样
- `R4A1`：仅距离向上采样
- `R1A4`：仅方位向上采样
- `R2A2`：距离向与方位向双向上采样

---

## 2. 取证节点说明

实验二使用如下取证节点：

- `Node-1 residual`：`channel_1bit - signal_up`
- `Node-2 RC`：距离压缩后的原始 `RC` 结果频谱图；指标表中的 `node2_rc` 仍基于下采样后的 `RC_crop`
- `Node-3 RCMC`：`RC_crop` 经过 `RCMC` 后的结果

需要特别说明：

- `Exp2_Node2_RC_Spectra.png` 当前展示的是**未下采样的原始 RC 结果**，因此四组图像尺寸不同，这是符合实验设计的。
- `Exp2_Mechanism_Metrics.csv` 中的 `node2_rc` 指标，仍是针对 `RC_crop` 计算的，因此它与 `node3_rcmc` 具有更强的可比性。

---

## 3. 指标定义

设待分析节点矩阵为 $X$，其二维频谱能量定义为：

$$
S_X(k_r, k_a) = \left| \mathcal{F}_2\{X\}(k_r, k_a) \right|^2
$$

代码中使用了 `fftshift(fft2(X))`，因此频谱中心已移到图像中心位置。设参考矩阵为 $X_{\mathrm{ref}}$，其支撑掩膜定义为：

$$
M(k_r, k_a) =
\begin{cases}
1, & \left|\mathcal{F}_2\{X_{\mathrm{ref}}\}(k_r, k_a)\right| \ge \tau \cdot \max\left|\mathcal{F}_2\{X_{\mathrm{ref}}\}\right| \\
0, & \text{otherwise}
\end{cases}
$$

其中本实验采用固定阈值：

$$
\tau = 0.35
$$

### 3.1 `off_support_ratio`

定义为二维总频谱能量中，落在参考支撑区外的比例：

$$
\mathrm{off\_support\_ratio}
=
\frac{\sum_{k_r,k_a} S_X(k_r,k_a)\,[1-M(k_r,k_a)]}
{\sum_{k_r,k_a} S_X(k_r,k_a)}
$$

物理意义：衡量该节点的二维频谱能量有多少泄漏到了有效支撑区之外。该值越小，说明频谱越集中，谱外污染越少。

### 3.2 `range_leakage_ratio`

当前版本不再使用“单条中心切片”，而是使用**整轴投影**。先对方位向求和，得到距离向边缘谱：

$$
P_r(k_r) = \sum_{k_a} S_X(k_r, k_a)
$$

对应的距离向有效支撑定义为：

$$
M_r(k_r) = \max_{k_a} M(k_r, k_a)
$$

则距离向泄漏比例为：

$$
\mathrm{range\_leakage\_ratio}
=
\frac{\sum_{k_r} P_r(k_r)\,[1-M_r(k_r)]}
{\sum_{k_r} P_r(k_r)}
$$

物理意义：衡量从距离向整体投影来看，有多少能量落到了参考距离支撑外。该值越小，说明距离向能量更集中。

### 3.3 `azimuth_leakage_ratio`

先对距离向求和，得到方位向边缘谱：

$$
P_a(k_a) = \sum_{k_r} S_X(k_r, k_a)
$$

对应的方位向有效支撑定义为：

$$
M_a(k_a) = \max_{k_r} M(k_r, k_a)
$$

则方位向泄漏比例为：

$$
\mathrm{azimuth\_leakage\_ratio}
=
\frac{\sum_{k_a} P_a(k_a)\,[1-M_a(k_a)]}
{\sum_{k_a} P_a(k_a)}
$$

物理意义：衡量从方位向整体投影来看，有多少能量落到了参考方位支撑外。该值越小，说明方位向能量更集中。

### 3.4 关于指标修订的说明

早期版本的 `azimuth_leakage_ratio` 使用“二维谱单条中心行切片”计算。该定义在本实验数据上会退化：由于参考支撑掩膜的中心行恰好为空，导致 `azimuth_leakage_ratio` 机械地恒等于 `1`。因此当前版本已改为整轴投影定义，上述数学公式与最新 `Exp2_Mechanism_Metrics.csv` 保持一致。

## 4. 最新数值结果

`Exp2_Mechanism_Metrics.csv` 的最新结果如下。

### 4.1 Node-1 residual

- `R1A1_NoUp`：`off = 0.8176903`，`range = 0.1997731`，`azimuth = 0.03822016`
- `R4A1`：`off = 0.8176916`，`range = 0.1998929`，`azimuth = 0.03821957`
- `R1A4`：`off = 0.8176839`，`range = 0.1997738`，`azimuth = 0.03849902`
- `R2A2`：`off = 0.8176866`，`range = 0.1998468`，`azimuth = 0.03841251`

结论：

- 四组方法在 `Node-1 residual` 上几乎重合；
- 说明“1-bit 残差刚产生时”的二维频谱泄漏差异并不显著；
- 双向上采样的优势并不是在这一节点就已经被大幅拉开。

### 4.2 Node-2 RC

- `R1A1_NoUp`：`off = 0.9260082`，`range = 0.4710436`，`azimuth = 0.1017848`
- `R4A1`：`off = 0.8939543`，`range = 0.3680131`，`azimuth = 0.07047553`
- `R1A4`：`off = 0.9010184`，`range = 0.3944266`，`azimuth = 0.07225090`
- `R2A2`：`off = 0.8928570`，`range = 0.3637502`，`azimuth = 0.06668038`

按照指标大小规律：

- `off_support_ratio`：`R2A2 < R4A1 < R1A4 < R1A1_NoUp`
- `range_leakage_ratio`：`R2A2 < R4A1 < R1A4 < R1A1_NoUp`
- `azimuth_leakage_ratio`：`R2A2 < R4A1 < R1A4 < R1A1_NoUp`

如果以 `R1A1_NoUp` 为基准，则 `R2A2` 在 `Node-2 RC` 相比 NoUp 的下降幅度约为：

- `off_support_ratio`：下降约 `3.36%`
- `range_leakage_ratio`：下降约 `22.78%`
- `azimuth_leakage_ratio`：下降约 `34.49%`

这说明双向上采样在 RC 节点后已经表现出最强的二维泄漏抑制能力。

### 4.3 Node-3 RCMC

- `R1A1_NoUp`：`off = 0.9260082`，`range = 0.4710436`，`azimuth = 0.1017848`
- `R4A1`：`off = 0.8939543`，`range = 0.3680132`，`azimuth = 0.07047553`
- `R1A4`：`off = 0.9010184`，`range = 0.3944266`，`azimuth = 0.07225092`
- `R2A2`：`off = 0.8928570`，`range = 0.3637502`，`azimuth = 0.06668039`

结论：

- `node2_rc` 与 `node3_rcmc` 数值几乎完全一致；
- 当前 leakage 指标对 `RCMC` 的敏感性较弱；
- 一个重要原因是本实验在 `RCMC` 之前先执行了 `RC_crop = two_dim_downsample_fft(RC_raw, ...)`，这会先裁掉一部分上采样带来的扩展频谱差异；
- 因此不能据此声称 `RCMC` 在该指标上又带来了显著改善。

## 5. 频谱图现象解释

### 5.1 `Exp2_Node1_Residual_Spectra.png`

该图展示的是 1-bit 量化残差的二维频谱形状。其主要现象为：

- `R1A1_NoUp`：高能区几乎占据原始支撑范围；
- `R4A1`：高能区在距离向扩展后呈现更窄的横向带状结构；
- `R1A4`：高能区在方位向扩展后呈现更窄的纵向带状结构；
- `R2A2`：高能区更接近居中的二维矩形块。

这说明上采样提供了额外的二维频域缓冲区，使量化残差不必完全挤在原支撑内。双向上采样能够同时在两个维度上提供更均衡的频域冗余。

### 5.2 `Exp2_Node2_RC_Spectra.png`

当前图像展示的是**原始 RC 结果**的二维频谱，而不是下采样后的 `RC_crop`。图中可以观察到：

- 四组方法在方位向上普遍仍较为分散；
- 距离向上出现有规律的黄蓝条纹；
- `R2A2` 与 `R4A1` 的高能区域更集中，`R1A1_NoUp` 的谱外扩散最明显。

物理解释如下：

1. `RC` 只沿距离向执行匹配滤波，方位向尚未聚焦，因此方位向能量仍表现为散射状态；
2. 距离向条纹主要来自距离匹配滤波器对距离谱的重塑；
3. 上采样方案之间的差别，不在于是否出现条纹，而在于条纹能量是否更集中于有效支撑附近。

---

## 6. 本实验当前能够支持的结论

基于最新指标和图像现象，实验二当前可以支持以下结论：

1. 双向上采样 `R2A2` 在 `RC` 与 `RCMC` 节点上都表现出最低的二维支撑外泄漏；
2. 相比 `NoUp`，上采样方案整体都降低了谱外泄漏；
3. 双向上采样的有效性可以从“二维频域冗余更充分、频谱支撑外泄漏更少”这一角度得到解释；
4. 双向上采样的优势在 `Node-1 residual` 并不明显，但在 `RC` 之后被明显放大。

---

## 7. 当前还不能过度声称的结论

本实验目前**不宜直接声称**：

1. `RCMC` 在当前 leakage 指标上又进一步显著改善了频谱结构；
2. “仅距离向上采样只改善距离向、仅方位向上采样只改善方位向”这一更细粒度方向性故事已经被完全验证。

原因是：

- `node2_rc` 与 `node3_rcmc` 数值几乎完全一致；
- 本实验样本上 `R4A1` 在三个指标上都略优于 `R1A4`，并没有形成非常干净的“单向对应单向”分离现象。

因此，更稳妥的结论应该是：

**双向上采样能够带来最均衡、最强的二维频谱泄漏抑制；单向上采样也有帮助，但当前单样本结果尚不足以严格证明“距离向只对应距离改善、方位向只对应方位改善”的更强机制叙事。**

---

## 8. 总结

实验二目前已经形成了一条较完整的证据链：

- 在 Node-1 残差阶段，四组方法差异较小；
- 在 RC 节点后，差异被明显放大；
- `R2A2` 在 `off_support_ratio`、`range_leakage_ratio`、`azimuth_leakage_ratio` 三项指标上均取得最优结果；
- 这说明双向上采样的主要收益来源，可以解释为：**通过增加二维频域冗余，降低量化误差在有效支撑外的扩散，并在后续成像处理中保留这种优势。**

该结论能够作为论文中“实验二机制解释”的主体结论使用，但对于 `RCMC` 的进一步作用和更细粒度的方向性归因，后续仍可继续补充更敏感的指标或更多样本验证。
